# Value Network Diagnostic（价值网络诊断）

来自《创新者的窘境》克莱顿·克里斯坦森。价值网络是你企业生存的「引力场」——它决定了什么项目能得到资源、什么创新会被系统性地扼杀。

## 核心理念

> "价值网络的异常强大的影响力，体现在阻碍企业向低端市场移动。"

你的价值网络由三部分构成：你的客户要求什么、你的成本结构允许什么、你的竞争对手在抢什么。这三个力合在一起，比你 CEO 的意志更强大。

## 诊断框架

### 维度 1：客户引力
- 你的主要客户是谁？他们最看重什么？
- 他们的需求在向什么方向演变？
- 有没有一群「你不想服务的客户」正在被别人服务？

### 维度 2：成本结构引力
- 你的毛利率要求是多少？低于多少的项目会被砍？
- 你的固定成本结构决定了你必须做什么规模的生意？
- 一个年营收 5000 万的新业务在你的组织里能活下来吗？

### 维度 3：竞争引力
- 你的竞争对手在争夺什么？
- 他们是不是也在向上走，留下低端空档？
- 有没有新进入者在做你们都不屑做的事？

## 经典症状

如果出现以下症状，说明价值网络正在限制你：

- 「这个市场太小，不值得做」→ 但小市场正在变大
- 「客户不需要这个功能」→ 但非客户需要
- 「利润率太低，不符合我们的要求」→ 但破坏者的成本结构完全不同
- 「我们的品牌不适合做低端」→ 所以你把低端市场拱手让人

```

- 被排除的商业机会：
- 被系统性拒绝的创新类型：

### 突破口
- 建议 1：
- 建议 2：
```
