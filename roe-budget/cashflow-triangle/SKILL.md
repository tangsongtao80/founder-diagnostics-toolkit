# 现金流三角诊断

来自《ROE预算管理：培养经营干部》付小平。核心框架：三种现金流各司其职——经营造血、投资抽血、融资输血。三角失衡，企业就出问题。

## 核心理念

> "经营现金流负责造血，投资现金流负责抽血，融资现金流负责输血。"

三种现金流就像人体的三个循环系统。任何一个出问题，都会拖垮另外两个。

## 三种现金流的健康标准

### 经营现金流（造血）
- **理想状态：正** — 企业经营本身在产生现金
- **危险信号：负** — 卖得越多、亏得越多，靠外部输血维持
- **诊断：** 是应收太大、库存太多、还是利润本身是假的？

### 投资现金流（抽血）
- **理想状态：负** — 企业在持续投入扩大产能/能力
- **危险信号 1：负得太多** — 投资过度，自我造血跟不上
- **危险信号 2：正** — 可能是在变卖资产维持生存

### 融资现金流（输血）
- **理想状态：视阶段而定** — 成长期可以正（融资），成熟期应趋于零或负（还债/分红）
- **危险信号：持续正** — 如果经营现金流为负，融资现金流持续为正 = 靠输血活着，随时可能断

## 八种现金流组合判断

| 经营 | 投资 | 融资 | 状态 | 含义 |
|:---|:---|:---|:---|:---|
| + | - | + | 🟢 健康成长期 | 自造血 + 在投资 + 有融资支持 |
| + | - | - | 🟢 健康成熟期 | 自造血 + 在投资 + 还债分红 |
| + | + | - | 🟡 收缩期 | 可能变卖资产还债 |
| - | - | + | 🟡 烧钱期 | 靠融资活着，必须确保跑道够长 |
| - | + | - | 🔴 危险 | 靠卖资产维持经营 |
| + | + | + | 🟡 囤积 | 有钱但不知道投哪 |
| - | - | - | 🔴 危机 | 三线全负，现金在枯竭 |
| - | + | + | 🔴 濒危 | 经营不造血 + 靠卖资产 + 靠融资 |
