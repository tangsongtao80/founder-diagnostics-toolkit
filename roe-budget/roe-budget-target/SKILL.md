# ROE 预算目标设定

来自《ROE预算管理：培养经营干部》付小平。核心主张：预算不是拍脑袋的数字游戏，而是从 ROE 倒推的系统工程——先定 ROE 目标，再拆解到每个经营动作。

## 核心理念

> "管理干部面向局部，而经营干部面向全局。"

预算的本质不是控制费用，而是把战略翻译成数字，再把数字拆解成每个人的动作。ROE 是这个翻译引擎的原点。

## 从 ROE 到预算的四步法

### 步骤 1：定 ROE 目标
- 当前 ROE 是多少？
- 目标 ROE 是多少？（对标行业标杆、考虑资本成本）
- ROE 的提升空间在哪：利润率、周转率、还是杠杆？

### 步骤 2：拆解到三因子
把 ROE 目标拆解为：
- 净利润率目标 → 需要多少收入、控制多少成本
- 总资产周转率目标 → 需要多少收入、控制多少资产
- 权益乘数目标 → 资产负债率控制在什么水平

### 步骤 3：拆解到经营动作
把因子目标进一步拆解为可执行的经营指标：
- 利润率 → 各产品线收入目标、毛利率目标、费用率上限
- 周转率 → 库存周转天数、应收账款天数、固定资产利用率
- 杠杆 → 有息负债上限、融资成本上限

### 步骤 4：分配到责任人
每个经营指标必须有人认领：
- 「净利润率 15%」→ 销售副总认收入、生产副总认成本、CFO 认费用
- 「周转率 2 次」→ 供应链认库存、销售认回款

## 预算目标的质量检验

目标定完后，检查以下四点：

- [ ] ROE 目标有明确的提升路径吗？（不能是「希望变好」）
- [ ] 每个因子目标都有责任人吗？
- [ ] 责任人知道他要做什么才能达成吗？（不能只看数字）
- [ ] 预算和实际有月度追踪机制吗？
