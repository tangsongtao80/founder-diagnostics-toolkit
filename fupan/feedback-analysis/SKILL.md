# 反馈分析法

来自《复盘》沈磊，源自彼得·德鲁克。一个简单但极其有效的方法：做重要决策时事先写下预期，9-12 个月后对比实际结果——让你看清自己真正擅长什么、不擅长什么。

## 核心理念

> "若想发现自己的长处，唯一的途径就是反馈分析法。每当做出重要决定或采取重要行动时，事先记录下自己对结果的预期。9～12 个月后，再将实际结果与预期比较。"

德鲁克本人用了 15-20 年，每次都有意外收获。这不是算命，是用数据认识自己。

## 适用场景

- 个人：职业选择、能力发展、搞清楚自己擅长什么
- 团队：关键决策的事后检验、培养判断力
- 组织：重大战略决策的长期追踪

## 执行步骤

### 步骤 1：记录决策和预期
每次做重要决定时，写下：
- 决策内容（做了什么选择）
- 预期结果（你认为会发生什么，越具体越好）
- 预期依据（你为什么这么认为）
- 记录日期

### 步骤 2：设定期限
根据决策的性质设定回看时间：
- 用人决策：6-9 个月
- 战略决策：12 个月
- 个人发展决策：12 个月

### 步骤 3：对比实际结果
到期后，拿出当初的记录，对比：
- 什么和预期一致？
- 什么和预期不一致？（这是最有价值的部分）
- 为什么判断对了/错了？

### 步骤 4：提炼模式
积累 5-10 个决策记录后，寻找模式：
- 你在什么类型的决策上判断最准？
- 你在什么类型的决策上反复出错？
- 你的盲区是什么？

### 步骤 5：调整行动
基于发现的模式：
- 把精力集中在你的长处
- 在盲区上寻求他人帮助或搭建系统
- 避免做你一再判断失误的决策类型

## 诊断输出格式

```
## 反馈分析记录

### 决策记录
- 决策：
- 预期结果：
- 预期依据：
- 记录日期：
- 回看日期：

### 回看对比（到期后填写）
- 实际结果：
- 与预期一致的地方：
- 与预期不一致的地方：
- 判断对/错的原因：

### 模式提炼（积累5条以上）
- 我的判断力强项：
- 我的判断力盲区：
- 应该多做的决策类型：
- 应该避免的决策类型：
```

## 注意事项

- 必须事先写——事后回忆会美化自己的判断
- 不需要分享给任何人，这是你的私人实验室
- 第一个 12 个月可能什么都看不出来，坚持 3 年以上才有模式
- 德鲁克说：大部分人在「人际关系」上的判断最差

## 来源

《复盘》，沈磊，第 2 章「德鲁克反馈分析法」；彼得·德鲁克《Managing Oneself》。
