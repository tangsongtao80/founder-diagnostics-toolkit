# Actionable vs Vanity Metrics（可执行指标 vs 虚荣指标）

来自《精益创业》埃里克·莱斯。不是所有数据都有用——虚荣指标让你感觉良好但无法指导决策，可执行指标能告诉你下一步该做什么。

## 核心理念

> "总注册用户数 100 万——这个数字除了让你在路演 PPT 上好看，什么决策都做不了。"

莱斯用「3A 原则」区分两类指标：可执行（Actionable）、可访问（Accessible）、可审计（Auditable）。

## 虚荣指标的特征

- 只升不降（如累计注册用户数、累计下载量）
- 不能帮你做决策（看完不知道下一步该做什么）
- 容易操纵（换个统计口径就变了）
- 伪装成宏大叙事（「我们已经有 X 万用户了」）

## 可执行指标的特征（3A）

### Actionable（可执行）
看完这个指标，你知道该做什么。比如「注册转化率从 10% 降到 5%」，你应该去排查注册流程。

### Accessible（可访问）
团队每个人都理解这个指标的含义，能用简单的报表看到。不是藏在分析师电脑里的复杂模型。

### Auditable（可审计）
数据来源清晰，可以追溯——你说转化率 5%，我能顺着漏斗验证。

## 常见替换对照

| 虚荣指标 | 替换为可执行指标 |
|:---|:---|
| 累计注册用户数 | 周活跃用户数 + 周增长趋势 |
| 总下载量 | 次日留存率 + 7日留存率 |
| 页面浏览量 | 用户完成核心任务的比例 |
| 媒体报道数量 | 媒体报道带来的注册转化率 |
| 累计收入 | 月度经常性收入（MRR）+ 增长率 |
| 粉丝数 | 互动率 + 来自粉丝的转化率 |

## 执行步骤

### 步骤 1: 列出当前使用的所有指标
不加筛选地列出团队正在看的所有数字。

### 步骤 2: 用 3A 原则逐一审查
每个指标问三个问题：
- 看完这个数字，我能做出什么具体行动？
- 团队里的任何一个人都能理解并获取这个数据吗？
- 我能追溯这个数字是怎么算出来的吗？

三个全是 YES = 可执行指标；任何一个 NO = 至少部分虚荣。

### 步骤 3: 建立可执行指标体系
为产品/业务设定 3-5 个核心可执行指标。

### 步骤 4: 建立报表
确保这些指标每周/每天自动更新，团队全员可见。

## 输出格式

```
## 指标审查结果

### 当前指标清单
| 指标名称 | 可执行？ | 可访问？ | 可审计？ | 判定 | 建议 |
|:---|:---|:---|:---|:---|:---|
| | ✅/❌ | ✅/❌ | ✅/❌ | 可执行/虚荣 | 保留/替换 |

### 新的可执行指标体系
- 北极星指标：
- 驱动指标 1：
- 驱动指标 2：
- 诊断指标：
```

## 注意事项

- 虚荣指标不是没用——对外宣传（PR、融资 PPT）可以用，但内部决策不能用
- 同一指标在不同阶段可能是虚荣的也可能是可执行的——判断标准永远是「能不能指导行动」
- 从虚荣指标转向可执行指标的过程通常会暴露真相，要做好心理准备
